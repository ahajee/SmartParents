Project Title : SmartParents - Cybersecurity Awareness Web Application for Parents

Description: A web application that contains cybersecurity related topics such as threats and controls for parents and children along with quiz and forum discussion feature that can be attempt by registered users. 

database: database/smartparent.sql 
Content:
  - index page
  - parents (threat, control)
  - children (threat->age range, control -> age range)
  - principle of islamic parenting 
  - register/ sign in
  - user page
  - forum page ( forum post display, add post, reply post)
  - quiz 
  - user profile update 
Run: this project use Xampp to be able to run on the browser
    : database script can be found in database folder
